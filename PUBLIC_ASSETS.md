Title: Public assets for page backgrounds

Place the images you want used for page backgrounds under `public/assets/` with these filenames:

- `home-hero.jpg` — Home / landing page
- `profile-hero.jpg` — Profile / Studio
- `builder-hero.jpg` — Room builder page
- `explore-hero.jpg` — Explore / gallery pages
- `uploads-hero.jpg` — Uploads page
- `auth-hero.jpg` — Login / Signup

If you want different file types, update the URLs in `app/globals.css` accordingly.

Example path in the repo after adding images:

public/assets/home-hero.jpg
